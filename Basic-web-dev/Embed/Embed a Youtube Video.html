How to Embed a YouTube Video to a Website

Follow this steps:

Step 1: Find a Video

Go to YouTube and choose a video you'd like to embed.

Step 2: Click the Share Button

Under the video, click the Share button to open the sharing options.

Step 3: Click Embed

Click the embed option to get the HTML code.

Step 4: Copy the Embed Code

Click the “Copy” button to copy the HTML iframe code..

Step 5: Paste into Your HTML File

Open your HTML file in a code editor and paste the embed code .

Result: Embedded Video
